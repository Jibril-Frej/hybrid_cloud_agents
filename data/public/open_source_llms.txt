Open Source Large Language Models

The open-source LLM ecosystem offers a wide range of models for on-premises deployment.
Qwen2.5 by Alibaba Cloud is a family of instruction-tuned models ranging from 0.5B to 72B
parameters. The 0.5B variant can run on commodity CPUs and is well suited for development
and resource-constrained environments.

BM25 (Best Match 25) is a classic probabilistic ranking function widely used in information
retrieval. It scores documents based on term frequency and inverse document frequency (TF-IDF),
with length normalization controlled by the parameter b and term saturation by k. BM25 is
computationally efficient compared to dense embedding approaches and performs well on
keyword-rich queries.

ChromaDB is an open-source vector store that supports both dense and sparse (BM25) embeddings.
Its Python client allows in-process or server-mode operation, making it suitable for
development, testing, and production deployments alike.

FastAPI is a modern Python web framework built on Pydantic and Starlette. It provides
automatic OpenAPI documentation, request validation, and high performance thanks to its
ASGI architecture.
